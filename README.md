### Clone local

Voici la marche à suivre pour lancer le projet en local : 

1. Clone the repo
   ```sh
   git clone https://github.com/LoicChazeau/Matheo---Chatbot-educatif
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the project
   ```sh
   npm run dev
   ```
4. Go to : http://localhost:5173/
